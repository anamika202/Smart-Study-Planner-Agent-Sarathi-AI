## Overview

Sarathi AI helps students convert long-term academic goals into manageable daily and weekly plans using AI-assisted planning and structured study workflows.

## Key Features

- Goal-based study planning
- AI assisted schedule generation
- Structured daily & weekly learning roadmap
- Student-friendly human-centered interface
